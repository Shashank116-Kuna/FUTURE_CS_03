# FUTURE_CS_03  
📘 API Security Risk Analysis – JSONPlaceholder
📌 Overview

This project presents a read-only API security risk analysis conducted on the JSONPlaceholder API.

The goal was to identify common API security risks and document them in a professional security report.

🎯 Objectives
Analyze publicly accessible API endpoints
Identify security weaknesses
Assess risk severity and business impact
Provide remediation recommendations
🧪 Tools Used
Postman (API testing)
Browser DevTools
Google Docs / Microsoft Word
GitHub (documentation)
🔍 Scope & Methodology
Target: https://jsonplaceholder.typicode.com
Testing Type: Read-only only
No exploitation or service disruption performed

Methods:

Endpoint testing
Manual inspection
Response analysis
⚠️ Key Findings
Missing authentication on endpoints
Public data exposure
No rate limiting
Predictable resource identifiers
Missing security headers
📊 Deliverables
📄 API Security Risk Analysis Report (PDF)
📸 Screenshots of API testing
📘 Documentation (this repository)
